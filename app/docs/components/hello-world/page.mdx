# Hello World

Lets get started making your first mockup component.

---

## 1. Setup basic project

Create new React project, here we will be creating a basic NextJS project with
tailwind css. You can do that by running `npx create-next-app@latest` command.
Know more at [NextJS.org](https://nextjs.org)

---

## 2. Create a file

After having a project setup, create a folder called `components` and in it
create a file called `MacbookMockup.tsx` or .jsx if you are using plan JS.

---

Copy the following minified code and paste it in the file you just created.
